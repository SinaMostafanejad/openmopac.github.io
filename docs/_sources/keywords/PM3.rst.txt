``PM3`` 
========

The PM3 methodÂ [`4 <references.html#pm3_1>`__] is to be used.

The PM3(tm) method is not in MOPAC, but it is so similar to PM3 that
PM3(tm) calculations can be run using the EXTERNAL option. To run
PM3(tm) calculations, use keywords PM3 and EXTERNAL=<filename>. The
contents of the file <filename> would be the normal PM3(tm) parameters.
An example of such a parameter set is as follows:
